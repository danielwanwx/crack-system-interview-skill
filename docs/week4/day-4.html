Senior / Staff pressure test

先答，再看完整 Q&A

真实追问 1

如何防止帖子已经对粉丝可见，但它引用的视频仍在转码？

触发：候选人把上传成功直接等同于发布成功。

先修：画 UploadSession、MediaVersion 与 Post 三个状态机，标出唯一可见性切换点。

真实追问 2

四 GB 上传在 95% 时断线，续传协议需要保存哪些证据？

触发：候选人只说 multipart，却无法判断哪些 part 可以信任。

先修：写出 create、uploadPart、listParts、complete 四个接口及幂等键。

真实追问 3

为什么 Feed API 不应该代理图片和视频字节？

触发：候选人让应用服务承担媒体上传和下载，导致带宽耦合。

先修：把 Feed JSON、授权 token、CDN cache key 与对象键分别画出。
