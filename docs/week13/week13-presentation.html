s7 · 1-D DP

1-D DP 算法

状态、递推、base case、循环方向与空间压缩。

LeetCode 
# House Robber: prev1 = best through i-1
prev2 = prev1 = 0
for money in nums:
    prev2, prev1 = prev1, max(prev1, prev2 + money)
return prev1

# Coin Change: dp[a] = min coins for amount a; dp[0]=0
回答模板

RECURRENCE

  • dp[i] 用自然语言定义：处理到 i 的最优/可行性/方案数。
  • 从“最后一步/是否取 i”推出转移。
  • base case 覆盖空输入与不可能状态。
  • 只依赖 i−1/i−2 才能安全压缩。
高频陷阱

INVARIANT

  • House Robber 更新前保存 old prev1。
  • Coin Change 无解是 infinity，不是 0。
  • Word Break 的 dp[i] 是 prefix 可分割。
  • LIS tails 不是实际序列。
面试锚点答案必须落点追问
Climbing Stairsways[i]=ways[i−1]+ways[i−2]O(n)/O(1)
House Robber II拆 [0..n−2] 与 [1..n−1]n=1
Decode Ways一位 + 合法两位0 不可独立
Coin Changemin(dp[a−c]+1)unreachable
Word Breakany dp[j] && word[j:i]字典剪枝
LISdp[i]=1+max dp[j]O(n²) → O(nlogn)

Q. Coin Change 先说什么？

答题方向：dp[a] 表示凑出 a 的最少硬币，dp[0]=0，其余 infinity；对每个 amount 枚举 coin，最后处理无解。
