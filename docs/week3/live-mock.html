WhatsApp

完成一张 WhatsApp 端到端设计图：标出 消息、收件箱投递状态与设备确认、同步提交边界、主要读写路径、分片键与用户可见状态。

Pressure prompts

  • Redis Pub/Sub 在发布瞬间丢了一条消息，为什么在线用户最终仍能收到？
  • 同一用户有手机、平板和电脑，其中电脑离线一周；你的投递状态怎样建模？
  • 一台 Chat Server 维持一百万连接时突然宕机，怎样避免重连把剩余集群再次打垮？

FB Live Comments

完成一张 FB Live Comments 端到端设计图：标出 直播内评论日志与观众消费游标、同步提交边界、主要读写路径、分片键与用户可见状态。

Pressure prompts

  • 为什么直播评论选 SSE 而不是 WebSocket？这个选择会在哪些基础设施上失败？
  • 同一直播的观众散在两百台 Realtime Server 上，新评论怎样只送到相关机器而不是全局广播？

  • FB Live Comments · 1) How can we ensure comments are broadcasted to viewers in real-time?
  • FB Live Comments · 2) How will the system scale to support millions of concurrent viewers?
