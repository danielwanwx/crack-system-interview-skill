Q. 为什么不只用 Redis SETNX 锁座位？

答题方向：锁不保存 authoritative inventory 且会失效；数据库条件更新/版本才是库存 invariant。Redis 可辅助协调，不能替代售卖记录。

s7 · 2-D DP

2-D DP 算法

用 grid 含义、边界和填表方向控制复杂状态。

LeetCode 
# LCS: dp[i][j] = LCS length of a[:i], b[:j]
if a[i-1] == b[j-1]: dp[i][j] = dp[i-1][j-1] + 1
else: dp[i][j] = max(dp[i-1][j], dp[i][j-1])
# Edit distance: dp[i][0]=i; dp[0][j]=j
模板

STATE

  • 先用自然语言定义 dp[i][j]。
  • 画 i/j 轴，说明子问题与最后决策。
  • 第 0 行/列是 base case。
  • 空间压缩时保存 old diagonal。
题型

PATTERNS

  • LCS：equal diagonal，else top/left。
  • Edit Distance：replace/delete/insert。
  • Unique Paths：top+left。
  • Burst Balloons：最后戳 k 的 interval DP。

Q. 如何解释 2-D DP 不像背式子？

答题方向：先定义每格的子问题，解释最后决策如何拆到已完成格子，再用 row/column base case 证明边界。
