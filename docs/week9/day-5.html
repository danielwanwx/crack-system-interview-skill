今日唯一产出物

交付

一张包含 URL discovery、canonicalization、dedupe、host-aware frontier、fetch、parse、content storage 与 recrawl 的 Web Crawler 白板。

验收标准

  • frontier 持久化且按 host/domain 调度，worker 崩溃后任务可重新领取。
  • robots.txt、crawl-delay、并发上限和 backoff 在调度层强制执行。
  • URL 与内容去重、10B 规模估算和五天完成目标有具体数据结构与吞吐计算。
不通过怎么修

若用一个全局 queue 结束设计，补 host buckets、next_allowed_at、lease、attempt、canonical URL、content hash 和 dead-letter 状态。
