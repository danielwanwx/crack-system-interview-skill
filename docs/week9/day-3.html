验收标准

  • 帖子和权限是权威事实，搜索索引只是可重建投影。
  • 关键词、短语、recency 和 like count 对应明确索引结构，而非请求时全量排序。
  • 删除和权限收紧有比普通索引更新更快的 deny/final check。
