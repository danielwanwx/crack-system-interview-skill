今日唯一产出物

交付

一张可在 15 分钟讲完的 Bitly 初版白板：FR/NFR、POST /urls、GET /{short_code}、URLMapping schema、create path 与 redirect path。

验收标准

  • 每个 NFR 都量化并能指出它改变了哪一项设计。
  • 能从 GET /{short_code} 的访问路径推出 short_code 唯一索引和过期语义。
  • create 与 redirect 两条路径分开，权威数据库、缓存和返回状态码职责清楚。
不通过怎么修

如果一开始就画 Redis、Kafka 或分片，先删除这些盒子，只用单服务加 Postgres 重画可工作的基线，再逐项用瓶颈证据加回组件。
